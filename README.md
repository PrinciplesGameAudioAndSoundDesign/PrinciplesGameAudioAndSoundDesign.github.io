# Principles of Game Audio and SoundDesign

Below you'll find supporting code, documentation, and the [official repository](https://github.com/PrinciplesGameAudioAndSoundDesign/PGASD-2ndEdition) for the 2nd edition of "Principles of Game Audio and Sound Design," by [Jean-Luc Sinclair](https://college.berklee.edu/faculty/jean-luc-sinclair) and [David Cardona](https://linktr.ee/dcbapc), published by [Routledge](https://www.routledge.com/).
